## How to Use

### Before Starting

Open:

**Options → Controls → Key Binds**

Locate **Social Interaction Screen** and make sure it is **not** assigned to **P**.

Assign another unused key.

---

### Open Gamble Rig

Press **P** in-game.

---

### Configure Paper Game

1. Hold your first item.
2. Press **P**.
3. Select **Paper Game**.
4. Click **From Hand**.

Repeat the same process using **From Hand (B)** for the second item.

Return both items to their droppers and begin the game normally.

---

### Hotkeys

| Key | Action |
|------|--------|
| **P** | Open Gamble Rig Menu |
| **O** | Toggle Overlay |
| **H** | Switch Forced Winner |
| **G** | Disable Rigging |
